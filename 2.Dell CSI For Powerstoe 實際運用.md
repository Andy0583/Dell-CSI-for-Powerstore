### 1.建立Storage Class

```
apiVersion: storage.k8s.io/v1
kind: StorageClass
metadata:
  name: "sc-powerstore"
  namespace: "ps"
provisioner: "csi-powerstore.dellemc.com"
parameters:
  arrayID: "PS556db8bad5cd"
  csi.storage.k8s.io/fstype: "nfs"
  nasName: "andy-nas"
  allowRoot: "true"
reclaimPolicy: Delete
allowVolumeExpansion: true
volumeBindingMode: Immediate

```
